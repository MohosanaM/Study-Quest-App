 Security & Privacy
This project’s privacy framework was refined through the lens of the **Amnesty International Digital Security & Human Rights** framework, ensuring that user data protection aligns with global humanitarian standards. 

Key implementations include:
- **Data Minimization:** Only essential local data is stored.
- **Privacy-First Design:** User anonymity is prioritized in all leaderboard features.
- **Secure Architecture:** Built with security hardening to protect student progress and personal info.
